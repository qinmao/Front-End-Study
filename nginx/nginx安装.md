# nginx 安装
## windows (是非服务的控制台应用程序)
  - 安装(http://nginx.org/en/download.html)
 ```bash
    cd c:\
    unzip nginx-1.17.2.zip
    cd nginx-1.17.2
    start nginx
  ```
## linux(centos)
  ```bash
    # 可以先查看一下
    yum list | grep nginx 
    yum install nginx
   ```
## mac
  ```bash
   brew install nginx
  ```
