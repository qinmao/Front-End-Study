## url 输入到展现的过程
  - DNS 解析:将域名解析成 IP 地址
  - TCP 连接：TCP 三次握手
  - 发送 HTTP 请求
  - 服务器（web 框架）处理请求并返回 HTTP 报文
  - 浏览器解析渲染页面
  - 断开连接：TCP 四次挥手

## 页面的渲染过程
1. 解析HTML，构建DOM树
2. 解析CSS，生成CSS规则树
3. 合并DOM树和CSS规则，生成render树
4. 布局render树（Layout/reflow），负责各元素尺寸、位置的计算
5. 绘制render树（paint），绘制页面像素信息
