# 对象的常用方法
## 对象的遍历
  ```js
    Object.keys(obj).forEach(key=>{})
    Object.values(obj).forEach(value=>{})
    Object.entries(obj).forEach(([key,value])=>{})
  ```
## 对象合并
* Object.assign()
  ```js
    let target = { a: 1, b: 1 };

    const source1 = { b: 2, c: 2 };
    const source2 = { c: 3 };

    Object.assign(target, source1, source2);
    target // {a:1, b:2, c:3}

  ```
