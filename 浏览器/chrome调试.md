# chrome

## 接口重新请求
- 选中Network
- 点击Fetch/XHR
- 选择要重新发送的请求
- 右键选择 Replay XHR

## 在控制台快速发起请求
- 选中Network
- 点击Fetch/XHR
- 选择Copy as fetch
- 控制台粘贴代码
- 修改参数，回车搞定

## 复制JavaScript变量 
- 使用copy函数，将对象作为入参执行即可

## 截取一张全屏的网页
- 准备好需要截屏的内容（注意懒加载）
- 打开开发者工具，cmd + shift + p 执行Command命令
- 输入 Capture full size screenshot 按下回车
